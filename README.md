# AIBill 

基于 Qt Widgets + SQLite + DeepSeek API 开发的智能记账分析app。

## 功能

* 账单新增、编辑、删除
* 条件搜索
* 账单 CSV 导入
* 自动分类
* 防重复导入
* 消费占比饼图
* 消费排行
* DeepSeek AI 消费分析
* 月度统计

## 技术栈

* Qt 5.9
* C++
* SQLite
* QtCharts
* QNetworkAccessManager
* DeepSeek API

## 项目亮点

* 独立完成需求分析、架构设计与开发
* 引入 AI 能力实现消费行为分析
* 利用交易单号实现防重复导入
* 采用 Service + Database + UI 分层设计
